# 快速开始指南 (Quick Start Guide)

## 5分钟快速体验

### 前置要求
- Windows 7 或更高版本
- 一台网络摄像头或DVR/NVR设备（知道IP、端口、用户名、密码）

### 步骤 1: 注册控件（首次运行必需）

**Windows 10/11**:
1. 双击根目录的 `注册OCX控件.bat`
2. 在弹出的UAC对话框中点击"是"
3. 看到成功提示后点击"确定"

**手动注册**:
```cmd
# 以管理员身份运行命令提示符
cd path\to\JBNVSDK\bin
regsvr32 JBNVOCX.ocx
```

### 步骤 2: 打开项目

使用 Visual Studio 打开 `VideoPlayerDemo.sln`

### 步骤 3: 编译运行

1. 按 `F5` 或点击"开始"按钮
2. 应用程序窗口将打开

### 步骤 4: 连接设备

1. 输入设备信息：
   - 服务器IP: 例如 `192.168.1.100`
   - 端口: 通常是 `34567`
   - 用户名: 通常是 `admin`
   - 密码: 您的设备密码

2. 点击"连接并播放"按钮

3. 如果一切正常，您应该看到实时视频！

### 步骤 5: 测试功能

- 点击"截图"按钮可以保存当前画面
- 点击"断开连接"可以停止视频

## 常见问题快速解决

### ❌ 提示"无法加载控件"
**解决**: 运行 `注册OCX控件.bat`（需要管理员权限）

### ❌ 编译错误
**检查**: 项目属性 → 生成 → 平台目标应为 **x86**

### ❌ 无法连接
**检查**:
1. IP地址是否正确
2. 用ping测试网络: `ping 192.168.1.100`
3. 用户名密码是否正确
4. 防火墙是否阻止

### ❌ 视频黑屏
**解决**: 确保 bin 目录中的所有DLL文件都在程序运行目录
